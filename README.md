# Paradise Nursery Shopping Application

## Project Description
Paradise Nursery is a fully functional e-commerce shopping cart application built with React and Redux. Users can browse houseplants organized by category, add them to a shopping cart, and manage their cart items.

## Features
- Browse houseplants organized into categories
- Add plants to shopping cart
- View cart with total cost per item and overall total
- Increase or decrease item quantity
- Remove items from cart
- Checkout button with "Coming Soon" message
- Dynamic cart icon showing total number of items
- Responsive navigation bar

## Technologies Used
- React
- Redux Toolkit
- CSS3
